This project compares different machine learning classifiers on a single CSV dataset (like a Kaggle-style dataset). First, main.py reads data/dataset.csv, then src/data_preprocessing.py splits the data into X (features) and y (target column: loan_status). Because the dataset contains text categories (object columns), it converts those categories into numbers using pd.factorize() so that all models can learn from them. Next, it performs a train/test split (train and test data are kept in order using shuffle=False), and src/models.py trains 6 classifiers: Bernoulli Naive Bayes, Random Forest, Gaussian Naive Bayes, Decision Tree, Multinomial Naive Bayes, and KNN. After prediction, it calculates Accuracy, Precision, Recall, and F1 score for each classifier, then src/visualization.py creates plots (a line graph for the scores and a bar chart for F1). Finally, all results are saved in the outputs/ folder as images and results.txt, so you can easily show which model performed best.
